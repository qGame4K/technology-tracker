Как запустить проект
	1.	Клонируте репозиторий или скачай его в папку.
	2.	Установите зависимости (нужен Node.js 16+):
        npm install
    3.  Запустите приложение:
        npm start
    4.	Откроется страница http://localhost:3000 в браузере.

Как пользоваться
	Загрузка карты:
	•	При первом запуске вам предложат загрузить JSON-файл с roadmap.
	•	Используйте файл  newroadmap.json , который лежит в папке  public/ , или свой собственный файл такого формата который написан в проге
